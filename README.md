
# Calculadora IOS

Calculadora do IOS com suas funcionalidades básicas

## Rodando localmente

clone o repositório no computador e instale as dependências necessárias do projeto
```bash
    @emotion/react": "^11.14.0,
    @emotion/styled": "^11.14.0,
    @mui/material": "^7.1.1,
```

## Stack utilizada

**Front-end:** CSS, ReactJS

**Back-end:** JSX
